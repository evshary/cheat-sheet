# D-Bus
Official Website: https://www.freedesktop.org/wiki/Software/dbus/

Used in Desktop communication, providing an interface for different processes to communicate with each other.

One of IPC in Linux

# Usage
* build
```
sudo apt install libdbus-1-dev
make
```
* Monitor
```
dbus-monitor
```
* Run server
```
./server
```
* Run client
```
./client
```
