# USB

* `lsusb`: To show USB device in Linux
  - For more detail: `lsusb -tv`

# modules

* USB 1.0/1.1 Host controller: uhci_hcd
* USB 2.0 Host controller: ehci_hcd
* USB 3.0 Host controller: xhci_hcd
* USB storage: usb-storage
* USB mouse: usbhid
