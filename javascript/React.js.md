# React.js

# Usage

* Install npm
* Install create-react-app

```shell
npm install -g create-react-app
```

* Create a new project

```shell
create-react-app hello-world
```

* Connect to server `localhost:3000`

```shell
npm start
```

* Package all html & css & js

```shell
npm run build
```
